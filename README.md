# Project1

This repository contains the source code for Project1, including backend Python scripts and a React-based UI.

## Structure

- `api/` - FastAPI routers for assessment, authentication, quiz, document reading, email, and settings.
- `ui/` - React frontend for the project.
- Python scripts for quiz assessment, question generation, email, and more.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/phurpawangchuk/CS529_Project1.git
   ```
2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. (Optional) Set up the UI:
   ```bash
   cd ui
   npm install
   npm start
   ```

## Usage
- Run backend services using the provided Python scripts or FastAPI routers.
- Access the frontend via the `ui` directory.
